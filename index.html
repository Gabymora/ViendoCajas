Laboratoria

Laboratoria enseña a jóvenes mujeres a crear productos digitales conel propósito de insertarlas a la creciente economía digital del Perú. Es un periodo de 6 meses, nuestras coders aprenden:

  1. El lenguaje del futuro.

    La programación está detrás de todos los productos que usamos día a día. Nuestras coders aprenden HTML 5, CSS y Javascript, terminando preparads para crear infinitos productos y soluciones web.

  2. Aprender a aprender.

    Para triunfar como desarrolladoras web nuestras coders se convierten en autodidactas, descubriendo que no hay límites para seguir creciendo en este campo.
